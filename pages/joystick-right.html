Do you have a favorite button on the interface?

The controller has two tiny buttons in the middle that do different things. So there is Start on the right. Sometimes it opens a menu, and you can pause the game with it. Then there is a Select button. It’s a very vague button, you don’t know what it does. It has different functions in each game. It was interesting for me to discover what the select button does in each game. I would open up a game and press select for the first time and see what would happen. In some games it would record, in some it would open a separate menu. There were different things for different games.
